# 基于VUE框架搭建的一个外卖项目


> ## 用到技术
> 
> 1.   基于vue为框架，构建整个结构
> 2.   vue-router做页面路由跳转
> 3.   vuex做整个项目中全局属性管理以及操作
> 4.   mint-ui，导入弹框组件，并在引用配置里配置了按需打包
> 5.   vue-lazyload，图片loading组件
> 6.   better-scroll，做页面滚动滑动效果
> 7.   swiper,做主页轮播组件
> 8.   Sass,做为css扩展语言
> 9.   mockjs模拟后台数据，拦截ajax
> 10.  axios做为后台接口请求
